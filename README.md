# Maquina Team — Martial Arts Academy Website

A responsive, static website for Maquina Team (martial arts academy). Built with semantic HTML5, modern CSS3, Bootstrap 4.5, jQuery (Slim), and Font Awesome. Includes a hero carousel, reviews slider, mobile‑friendly navigation, and performance‑minded styles.

Live preview (optional): Enable GitHub Pages on this repository to serve the site from the main branch root.

## Tech Stack

- HTML5 (semantic structure)
- CSS3 (custom styles, responsive utilities)
- Bootstrap 4.5 (layout, grid, components)
- JavaScript (vanilla)
- jQuery Slim (carousel initialization and DOM helpers)
- Font Awesome (icons)

## Features

- Responsive layout (mobile‑first, Bootstrap grid)
- Hero carousel (auto‑rotate ~2.5s via Bootstrap’s carousel)
- Reviews/testimonials slider (lightweight custom logic)
- Mobile navigation with collapsible menu
- Visual accents and utility classes for quick UI iteration
- Ready for SEO and performance optimizations (see checklist)
